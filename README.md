# 📱 Responsive Navigation Bar (NavBar Responsiva)

> Um componente de navegação moderno e adaptável para aplicações web, desenvolvido com HTML5 e CSS3, focado em performance e experiência do usuário (UX).

## 📖 Sobre o Projeto

Este projeto faz parte do meu portfólio de estudos em **Programação Full-Stack**. O objetivo foi construir uma **NavBar Responsiva** que mantém a elegância e funcionalidade em diferentes tamanhos de tela. 

A solução foca em uma navegação limpa para desktop e um menu intuitivo para dispositivos móveis, garantindo que o usuário tenha acesso rápido às principais seções do site, como Início, Sobre e Serviços.

## ✨ Destaques e Diferenciais Técnicos

* **Lógica de Estado Nativa (Checkbox Hack):** Uso de `input type="checkbox"` e `label` para controlar a visibilidade do menu mobile puramente via CSS, reduzindo o tempo de carregamento e dependência de scripts.
* **Design Modular:** Estruturação semântica utilizando a tag `<nav>` e listas não ordenadas (`<ul>`), facilitando a manutenção e a adição de novos links de navegação.
* **Responsividade Integrada:** Preparado para se adaptar a diferentes resoluções de tela, garantindo a integridade visual da "Marca" e dos itens de menu em qualquer dispositivo.
* **Interface Intuitiva:** Implementação de classes de estado ativo (`.active`) para fornecer feedback visual de navegação ao usuário.
* **Organização Profissional:** Separação clara de responsabilidades, com a marcação estrutural no `index.html` e a estilização em uma folha de estilos externa vinculada.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica do componente de navegação.
* **CSS3:** Estilização de layout, animações de transição e gerenciamento de estados responsivos.
* **Font Awesome (v5.15.2):** Iconografia para o botão de menu (hambúrguer) de forma performática.
